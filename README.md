### BGP直连 vs 软银/IIJ：线路是灵魂

日本机房的线路大致分两类。一类是 **BGP 国际线路或中国直连优化线路**，走的是常规国际路由，国内三网访问都能直连，延迟稳定但带宽通常给得不夸张，适合建站、轻量中转这种"细水长流"的场景。另一类是 **软银（SoftBank）或 IIJ 这类高端运营商线路**，带宽大、晚高峰不容易掉速，适合跑大流量、看视频、做游戏加速这类"吃带宽"的活儿。

简单粗暴的判断法：**追求稳定和性价比选直连/BGP，追求大带宽和晚高峰表现选软银/IIJ**。

### 带宽、流量、原生IP：别被参数绕晕

- **带宽**：日本机房常见 100Mbps 起步，高端套餐能到 800Mbps 甚至 1Gbps。带宽越大，并发能力越强，但价格也越贵。
- **流量**：注意是"月流量"还是"不限流量"，超了要么限速要么加钱。
- **原生IP**：指的是机房所在地的本地 IP，对于解锁当地流媒体、做跨境业务识别很重要。日本原生 IP 在解锁日本区 Netflix、AbemaTV 等服务时更稳。
